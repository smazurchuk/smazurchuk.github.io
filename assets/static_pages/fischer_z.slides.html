Fischer Z Transform¶

We will start with making 200 correlated time series, and then seeing what happens when we Fischer Z transform the distribution

In [1]:
import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt
from sklearn.datasets import make_spd_matrix

plt.rcParams['figure.figsize'] = [6, 4]
plt.rcParams['figure.dpi'] = 200 

# Reference: https://math.stackexchange.com/questions/163470/generating-correlated-random-numbers-why-does-cholesky-decomposition-work
X = make_spd_matrix(200,random_state=42) # Generate positive-definite covariance matrix
L = np.linalg.cholesky(X)                # Choleskly decompose
time_series = np.matmul(L,np.random.rand(200,1000)) # Generate correlated time series
corr_mat = np.corrcoef(time_series)      # Create correlation matrix
sns.heatmap(corr_mat)

We see that it "normalizes" correlation coeficients! To quote wikipedia on the topic of fischer-transformation:

This is because, when the transformation is applied to the sample correlation coefficient, the sampling distribution of the resulting variable is approximately normal, with a variance that is stable over different values of the underlying true correlation.
